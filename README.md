Project Overview

This Unity C# project implements a Gemini AI Chat Editor Window, providing AI-assisted tools for managing and enhancing your project. Key features include:

AI-Assisted Code and Content Management
Analyze, optimize, and annotate C# code or other project files (JSON, CSV, etc.) directly within Unity.
Automatically generate explanations, detect errors, optimize performance, and translate content.
Project Memory (Global Context / GDD)
Store a Game Design Document or project summary that the AI remembers across all chat tabs.
AI suggestions and code generation are based on this context.
Multiple Tabs and Persona Modes
Create multiple chat tabs with different expertise modes: Standard, Software Developer, Game Designer, or Shader Expert.
Web Search Integration
Enable web search per tab so the AI can provide answers with up-to-date information.
Visual and Audio Input Support
Send Game View screenshots or microphone recordings to the AI for analysis.
Multi-File Handling and Code Saving
Drag-and-drop multiple files into the chat tab.
Save AI-generated code directly into the Unity project or overwrite existing files.
Error Analysis and Console Monitoring
Read the latest Unity Console errors and provide AI-generated solutions.
User-Friendly Interface and Quick Actions
Quick actions include code optimization, adding comments, bug detection, and translation.
AI responses are displayed in a scrollable panel for easy viewing and copying.

In summary, this system automates AI-assisted code generation, optimization, error checking, and context-aware suggestions, streamlining Unity development and enhancing productivity.
